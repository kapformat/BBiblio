Inscription Jury

Reglement :

L’admission d’un juré se fait par les administrateurs après validation d’un formulaire mis à disposition du public. Après acceptation, il se voit notifier par mail d’une confirmation ou refus de candidature. Dans le mail, il obtiendra sa clé de connexion composé d’un login et mot de passe.
